Instalação

Pré Requisitos

Para iniciar siga as etapas abaixo para instalar e configurar todos os pré requisitos necessários

  • Node.js

    Para utilizar o nosso protótipo e aproveitar ao máximo você irá precisar baixar e instalar o Node.JS. Caso não tenha o mesmo já instalado, você pode baixar o instalador do site oficial, recomendamos que baixe a versão LTS do Node.JS para garantir a melhor experiência possível.

    Download Node.js

Instalação

Para configurar, siga as etapas mencionadas abaixo:

Você pode executar os seguintes comandos para executar o projeto localmente ou compilar para uso em produção

Você pode executar esta aplicação seguindo o gerenciador de pacotes:Bun, Yarn or NPM, preferencialmente em Bun!

1. Bun

Caso você não tenha o bun instalado no seu PC, use o npm i -g bun ou sudo npm i -g bun para instalar

Comando Descrição
bun i
 Esse comando irá instalar todas a dependencias necessárias na pasta node_modules
bun dev Executa o projeto localmente e inicia o servidor de desenvolvimento
bun run build Irá deixar o seu aplicativo pronto para ser implantado
2. Yarn

Se você não tem o yarn instalado no seu PC, use o comando a seguir npm i -g yarn ou sudo npm i -g yarn

Comando Descrição
yarn
 Esse comando irá instalar todos as dependencias necessárias na pasta node_modules
yarn dev Executa o seu projeto localmente iniciando o servidor de desenvolvimento.
yarn build Irá deixar o seu aplicativo pronto para ser implantado
3. NPM

npm vem pré-instalado quando você instala o Node.JS

Comando Descrição
npm i ou npm i --force
 Esse comando irá instalar todos as dependencias necessárias na pasta node_modules
npm run dev Executa seu projeto localmente iniciando um servidor de desenvolvimento.
npm run build Irá deixar o seu aplicativo pronto para ser implantado
