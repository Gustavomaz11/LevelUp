Introdução

Template de administração feito com Bootstrap v5.3.3 e React v19, usando tecnologias web modernas como HTML5 e CSS3. Ele traz um painel bonito, vários componentes prontos, widgets e páginas super versáteis, fáceis de integrar em qualquer projeto.
Totalmente responsivo e altamente personalizável sendo uma solução poderosa desenvolvido pela Fundação de Saúde Parreiras Horta para criar CMSs e painéis administrativos. Além disso, a documentação irá ajuda-los. Aproveitem ao máximo todas as funcionalidades.

Caso você tenham alguma dúvida podem entrar em contato nos canais de comunicação disponibilizados durante a matéria de Residencia de Software.
