Customização

Personalizando a paleta de cores

Para personalizar a paleta de cores tudo que você precisa fazer é ajustar algumas variáveis SCSS Com essa modificação você pode criar um visual único e personalizável para seu tema que poderá refletir a identidade do que será criado.

Para modificar as cores nos temas existentes, você pode abrir o arquivo _theme-default.scss localizado em src/assets/scss/. Basta alterar o valor de qualquer variável para personalizar a paleta de cores do tema. Suas alterações serão refletidas automaticamente em qualquer elemento HTML ou componente baseado no Bootstrap.

Customizando a barra lateral esquerda, largura do layout e barra lateral direita

A FSPH oferece flexibilidade para personalizar a aparência e a funcionalidade da barra lateral esquerda, a largura total do layout e a barra lateral direita.

  • Você pode escolher entre diversos tamanhos, tema e tipos para barra lateral esquerda e até ocultá-la, se necessário
  • Também é possível definir a largura geral do layout e ativar/desativar a barra lateral direita

Para personalizar os temas existentes, abra o arquivo useLayoutContext.jsx em src/context/useLayoutContext.jsx e faça as alterações necessárias na configuração dentro do arquivo JavaScript.

Titulo Tipo Opções Descrição
theme String "light" | "dark" Altera o esquema de cores geral para o modo escuro
topbarTheme String "light" | "dark" Altera o esquema de cores da barra superior para um valor especificado.
menu : Theme String "light" | "dark" Altera o esquema de cores do menu para um valor especificado.
menu : size String "default" | "compact" | "condensed" | "sm-hover" | "full" | "fullscreen" | "sm-hover" | "sm-hover-active" O tamanho da barra lateral esquerda é reduzido para um tamanho comprimido (pequeno).

Adicionar nova página

Fornecemos uma página inicial localizada na pasta src/ que pode ajuda-lo a começar de forma rápida e fácil ao adicionar novas páginas.
